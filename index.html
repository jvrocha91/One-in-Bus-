Termos de uso

Este é um sistema para compras de passagens de transporte público. Ele é dividido em 4 principais setores, sendo eles:

Gerar bilhete - Aqui você gerará o seu bilhete primário (Seu ID), ele será vinculado com os demais bilhetes que poderão ser selecionados por você!

Recarga - Neste espaço, é possível escolher qual tipo de passagem você deseja, cada passagem possui funcionalidades diferentes, os tutoriais de uso de cada uma delas pode ser encontrado nesta mesma aba! Além de escolher a passagem, todos os pagamentos são realizados a partir da aba Recarga!

Bilhetes - Nesta área é possível encontrar as informações dos seus bilhetes válidos com suas respectivas informações como código, data de emissão, data de validade, preço, quantos usos ainda restam, e assim sucessivamente.

Histórico - Por último, na aba "Histórico" é possível verificar todas as ações realizadas com os bilhetes e/ou sistema, ou seja, se você comprou, utilizou ou deixou expirar algum bilhete. Dessa forma, nessa área, é possível administrar de uma melhor forma o uso dos seus bilhetes.

Este sistema não recolhe nenhum dado pessoal e/ou comprometedor além do necessário. Clicando no botão "ACEITO" abaixo, você concorda que para a utilização dos serviços, é necessária a inserção de formas de pagamentos para as recargas dos bilhetes. Destarte, todas as informações coletadas serão mantidas em total sigilo e serão utilizadas somente para melhor usufruto do sistema pelo usuário.
